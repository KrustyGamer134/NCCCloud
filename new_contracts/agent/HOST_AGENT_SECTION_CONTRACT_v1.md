# Host Agent Section Contract v1

Status: AUTHORITATIVE

Scope: `ncc-agent/`

Purpose: Defines machine-local execution ownership.

## 1) Agent Ownership

- host-local install and update work
- process spawn and stop work
- runtime inspection and reporting
- local filesystem and machine interaction

## 2) Agent Non-Ownership

- lifecycle legality
- frontend semantics
- auth and session logic
- supported-game catalog ownership

## 3) Rule

Agent executes approved work and reports results. It does not create independent product policy.
